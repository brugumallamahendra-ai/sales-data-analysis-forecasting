# 📊 Sales Data Analysis & Forecasting System

## 🎯 Problem Statement
Businesses need accurate sales forecasts to plan
inventory, budget and strategy.

## 🛠️ Tech Stack
- Python, Pandas, NumPy
- Scikit-learn, Matplotlib, Seaborn

## 🔍 What I Did
1. Collected and cleaned multi-source sales datasets
2. Performed Exploratory Data Analysis (EDA)
3. Did feature engineering to find patterns
4. Built regression models - Linear, Random Forest
5. Evaluated models using RMSE, MAE, R² Score
6. Created visual dashboards for business insights

## 📈 Result
Accurate sales predictions with clear visual
dashboards for business decision making.

## 🚀 How to Run
pip install pandas numpy scikit-learn matplotlib seaborn
jupyter notebook
